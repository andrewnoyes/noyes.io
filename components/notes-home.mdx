> Computer programming is, fundamentally, about two things:
>
>    - Problem-solving using computers
>    - Learning to control complexity while solving these problems
>
> *[Yes, and...](https://htmx.org/essays/yes-and/)*

> ...it is concluded that the proper, primary aim of programming is, not to produce
programs, but to have the programmers build theories of the
manner in which the problems at hand are solved by program
execution.
>
> *[Programming as Theory Building](https://ingenieria-de-software-i.github.io/assets/bibliografia/programming-as-theory-building.pdf)*
