> Semantic elements are elements with meaning. A semantic element clearly describes its meaning to both the browser and developer.

---

Semantic elements that can be used to define differents parts of page:

- `<article>`
- `<aside>`
- `<details>`
- `<figcaption>`
- `<figure>`
- `<footer>`
- `<header>`
- `<main>`
- `<mark>`
- `<nav>`
- `<section>`
- `<summary>`
- `<time>`
