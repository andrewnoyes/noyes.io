**Misc libraries that look cool:**
- [nuqs](https://github.com/47ng/nuqs): type-safe search params state manager - like useState, but stored in the URL query string
- [wouter](https://github.com/molefrog/wouter#faq-and-code-recipes): routing for react/preact
  - noticed this as a dependency on [la suite numerique's meet frontend](https://github.com/suitenumerique/meet/blob/main/src/frontend/package.json), and in addition to a great name, its API is more minimal and seems to encompass the featureset I'm generally looking for (and not everything that react-router offers 😛)
