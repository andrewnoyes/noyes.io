- [Delete all local branches except for develop/master/main](https://stackoverflow.com/a/26152200/3757297):
```bash
git branch | grep -v "develop" | grep -v "master" | grep -v "main" | xargs git branch -D
```
- Display the "short sha" and "short description" on single line:
```bash
git log --oneline
```
- [20 most changed files in last year](https://piechowski.io/post/git-commands-before-reading-code#what-changes-the-most): 
```bash
git log --format=format: --name-only --since="1 year ago" | sort | uniq -c | sort -nr | head -20
```
- [Every contributor ranked by commit count](https://piechowski.io/post/git-commands-before-reading-code#who-built-this): 
```bash
git shortlog -sn --no-merges
```
